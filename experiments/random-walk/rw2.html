Random Walk

Random walk simulations are used in science and economics to model nature or stock markets. On this site you can see a few that I have created for fun

.

Notes on the simulation

This simulation will stop after 25,000 steps. Each step is 2px long and randomly moves in one of four directions up, down, right or left.

The graphics are created using the HTML5 canvas element and the programming is done with JavaScript.
