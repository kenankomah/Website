Snake Rules and Details

The bars below will give you some information about the game. Click on them to either expand them for more information or to hide them again.

Show Rules
Aim: The aim of this game is very simple, it is to get the snake to eat its food and to make it grow as long you can.

However there are a couple of things to avoid, you should never run into any of the walls and never run into yourself if any of this happens then the snake will die and it will be game over.

How to play: You can control the snake using the arrow keys i.e down key to move down, right key to move right and so forth. Use the space bar as another way to pause or play.
More Details
Background: This snake game is based on the same basic idea as Nokia phone game from the 1990's.

Technical info: I used the HTML5 canvas element to create the graphics for the game. For the sound I used the JavaScript audio object.

The animation and programming was all done with JavaScript.
